Question-3: What is event delegation in the DOM?

We know that events bubble up from the target element to the top. In event delegation, we attach an event listener to the parent element and with it handle the events of it's child elements.

Question-4: What is event bubbling in the DOM?

Its a behaviour of dom where the event starts from the item where it's happening and keeps moving upto html tag (bubblin up) unless it's specifically stopped.
